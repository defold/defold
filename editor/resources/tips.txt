# Hidden gems

Did you know? You can type simple math expressions like 640 / 2 in numeric property fields.
Did you know? You can press Ctrl/Cmd+D to select the word under the cursor. Pressing again will select the next occurrence, and so on.
Did you know? You can edit files side by side: right-click on editor tab and select Move to Other Tab Pane.
Hold Alt and Drag to perform box selection in code editor!
To move an object along its rotated axis, select Local Space from the Edit menu.

# Tips

To avoid breaking changes in [Library dependencies](https://www.defold.com/manuals/libraries/#_library_url) it is recommended to always depend on a versioned release instead of the latest code from the master branch.
You can open the [Web Profiler](https://www.defold.com/manuals/profiling/#_the_web_profiler) from the Open Web Profiler menu option in the Debug menu.
Did you know? You can tweak your game [without stopping it](https://www.defold.com/manuals/hot-reload/) — on any device!
