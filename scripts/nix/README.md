# Nix

This script lets you build Defold on Nix

## Usage

You need [Nix or NixOS](https://nixos.org/download/)

Next run the nix shell using:

```bash
$ ./scripts/nix/run.sh
```

After that just use the `./scripts/build.py` scripts (except for shell) [as instructed in the build instructions](https://github.com/defold/defold/blob/dev/README_BUILD.md).
